# Wink Project

## Purpose
Project data for the Wink APN fiber route in West Texas (Winkler, Andrews, and Ward counties).

## Expected Files
- **route geometry** (.kmz or .kml) - Route geometry data
- **intersection data** (.csv) - Intersection reference data
- Additional project-specific data files as needed

## Usage
Add route and intersection data files to this directory for processing with the geocoding pipeline.

## Current Files
- `wink.kmz` - Route geometry (KML/KMZ format)
- `wink-intersection.csv` - Intersection reference data
